<h1 style="text-align:center">The AABs : ALL ABOUT</h1>
<span style="text-align:center; display:block">Luca G.</span>

<br>
<p>Git project designed to provide an interactive and user-friendly way to explore and test various JavaScript methods across different data types.  With just a simple console.log statement and the name of a constant, users can immediately test the syntax and behavior of different methods. </p>
